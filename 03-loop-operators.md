## Цикл for
### Теория
Пример
``` C++
#include <iostream>

using namespace std;

int main() {
    cout << "Before loop" << endl;
    for (int i = 0; i < 10; ++i) {
        cout << "Loop iteration: i = " << i << endl;
    }
    cout << "After loop" << endl;
}
```

Вложенные циклы
``` C++
#include <iostream>

using namespace std;

int main() {
    for (int i = 0; i < 10; ++ i) {
        for (int j = 0; j <= i; ++j) {
            cout << j << " ";
        }
        cout << endl;
    }
}
```
